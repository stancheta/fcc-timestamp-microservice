FCC API Basejump: Timestamp microservice

You are able to pass a string as a parameter and it will check to see if the string has either a natural language date (ex. May 13, 1994) or a unix timestamp. If it contains either of the two, it will return JSON for both. If not, it will return null for those properties.

Example Usage:

https://stancheta-timestamp.herokuapp.com/December%2015,%202015
https://stancheta-timestamp.herokuapp.com/1450137600

Example Output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
